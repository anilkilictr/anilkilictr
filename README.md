## 👨‍💻 About Me

I am a **System & Network Engineer** focused on enterprise networking, infrastructure operations, network security and IT automation.

My GitHub profile is designed as a technical portfolio where I document practical labs, configurations, troubleshooting notes and automation projects around real-world infrastructure scenarios.

My current technical focus includes:

- Enterprise network design, routing and switching
- FortiGate firewall, VPN and secure network architecture
- Windows Server, Active Directory, DNS, DHCP and GPO
- Docker-based self-hosted infrastructure
- Monitoring with Zabbix and Grafana
- PowerShell, Bash, Python and n8n-based IT automation
- AI-assisted IT operations and local LLM workflows
